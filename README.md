## Introduction
"Magic Memory Game" is a React.js web game designed to improve memory skills.

## Getting Started
1. Install dependencies:

   ```npm install```

2. Start the game:

   ```npm run dev```

Enjoy enhancing your memory with the "Magic Memory Game"!
